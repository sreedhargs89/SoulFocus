===========================
REQUIRED SCREENSHOTS
===========================

App Store requires screenshots for each device size you claim to support.
The MINIMUM required are:

  1. iPhone 6.9" (iPhone 16 Pro Max)  →  1320 × 2868 px
  2. iPhone 6.5" (iPhone 11 Pro Max)  →  1242 × 2688 px   (can reuse 6.9" if submitted)
  3. iPad 13"    (iPad Pro 13")       →  2064 × 2752 px   (required if iPad is supported)

You declared TARGETED_DEVICE_FAMILY = 1 (iPhone only), so you only need iPhone sizes.

TIP: Submit only the 6.9" size — Apple will scale it for other sizes automatically.

===========================
HOW TO TAKE SCREENSHOTS
===========================

1. Run the app on an iPhone 16 Pro Max Simulator (Xcode > Simulator)
2. Navigate to each screen listed above
3. Press Cmd+S in Simulator to save screenshot to Desktop
4. Use Figma, Canva, or Sketch to add headline text overlays on a branded background
